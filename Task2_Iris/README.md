# Iris Flower Classification 🌸

This project classifies iris flowers into three species using machine learning.

## 📌 Objective
To build a classification model that predicts the species of an iris flower based on its measurements.

## 📊 Dataset
The dataset includes:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width
- Species

There are three species:
- Iris-setosa
- Iris-versicolor
- Iris-virginica

## 🛠 Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

## ⚙️ Steps Performed
1. Load dataset
2. Data preprocessing
3. Train-test split
4. Model training using classification algorithm
5. Prediction and evaluation

## 📈 Output
The model predicts the species of iris flower based on the input measurements.
