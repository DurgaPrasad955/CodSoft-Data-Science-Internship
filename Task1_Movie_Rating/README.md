
# Movie Rating Prediction 🎬

This project predicts the rating of a movie based on different features such as genre, director, and actors using machine learning techniques.

## 📌 Objective
To build a machine learning model that predicts movie ratings using historical movie data.

## 📊 Dataset
The dataset contains information about movies including:
- Movie Name
- Year
- Genre
- Director
- Actors
- Duration
- Rating

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## ⚙️ Steps Performed
1. Data collection and loading
2. Data preprocessing and cleaning
3. Feature encoding
4. Model training using Linear Regression
5. Model evaluation

## 📈 Output
The model predicts movie ratings based on the given movie attributes.
